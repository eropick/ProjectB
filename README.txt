#기본 구현은 다 되었는데 한 번씩 테스트 해보시면 좋을 것 같아요

첫 번째는 브레이크 샷이고 이 다음에 들어가는 공부터 공의 개수에따라
자신의 공 타입이 정해져요.

마우스 오른쪽 버튼 누를 때 공이 포켓 들어가도록 테스트를 할 수 있는데
점수가 카운트 안되면 턴이 안 지난 상황입니다.

=> 카운트가 되기 전까지는 한 턴에 플레이어 볼을 쳐서 한 번에 공이 여러 개 들어간 상황을 의미합니다.
=> 카운트가 되면 다음 턴이 되었다는 의미입니다.

#테스트는 다음과 같이 진행해보시면 됩니다.

1. 브레이크 샷을 친다.
=> 플레이어 볼을 직접 건드려도 되고, 우클릭으로 공을 넣어도 됩니다.

2. 원하는 공을 포켓에 넣어서 타입이 지정되는지 확인

3. 타입이 결정되면 해당 타입의 공을 모두 넣고 점수가 카운트 될 때까지 확인

4. 마지막으로 8번공이 들어갔을 때 누가 승리했는지 확인.
